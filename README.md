# biogas-prediction
Dataset generated for the implemented models in the paper: Biogas prediction for a swine farm bio-digester using a surrogate model based on machine learning.
available at: https://www.mdpi.com/3865004

Official repository with detailed and well-organized information as well as instructions is available on IEEE dataport: https://ieee-dataport.org//documents/time-series-data-biogas-production-and-lag-based-prediction-swine-farm-bio-digester
